# 🔐 Password Generator

A simple and responsive Password Generator built using **HTML, CSS, and JavaScript**. It generates secure random passwords using letters, numbers, and special characters, with a one-click copy feature.

## ✨ Features

- 🔑 Generate secure random passwords
- 🔢 Includes uppercase & lowercase letters
- 🔣 Includes numbers & special characters
- 📋 One-click copy to clipboard
- 🎨 Clean and responsive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 What I Learned

- DOM Manipulation
- Event Listeners
- Functions
- `Math.random()`
- `Math.floor()`
- `for` loops
- String concatenation
- Clipboard API (`navigator.clipboard.writeText()`)
